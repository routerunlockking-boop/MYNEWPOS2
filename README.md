# Smart Zone LK - WiFi Router E-Commerce Store

## Overview
A complete, professional e-commerce website for selling WiFi routers and networking devices in Sri Lanka. Built with vanilla HTML, CSS, and JavaScript.

## Features

### 🛒 Customer Features
- **Homepage** with hero banner, trust badges, categories, featured products
- **Shop page** with filters (category, brand, WiFi standard, price range) and sorting
- **Product detail pages** with full specifications, image gallery, quantity selector
- **Shopping cart** with add/remove, quantity updates, promo codes
- **Checkout** with Sri Lankan districts, 3 payment methods (COD, Bank Transfer, Online)
- **User authentication** (Login/Register with Google & Phone options)
- **About, Contact, Shipping & Returns, Warranty** pages

### 🔐 Admin Dashboard
- **Secure login** (admin@smartzonelk.com / admin123)
- **Dashboard** with revenue stats, orders, sales chart, low-stock alerts
- **Product management** - Add, Edit, Delete with full specifications
- **Order management** - View all orders, update status (6 statuses)
- **Customer list** with order history and spending
- **Analytics** - Top products, conversion rates, page views

### 🎨 Design
- **Color scheme**: Blue (#0066FF), Green (#00C853), Dark (#0A0E27), Cyan (#00E5FF)
- **Fonts**: Inter (main), JetBrains Mono (technical elements)
- **Mobile-first responsive** design
- **Smooth animations** and hover effects
- **Toast notifications** for all actions
